This is a template blog post that you can use as a starting point for creating your own blog posts.

## How to Create a New Blog Post

1. Copy this file in the `_posts` folder
2. Rename it following the format: `YYYY-MM-DD-your-post-title.md`
3. Update the front matter (the section between the `---` markers):
   - **title**: Your post title
   - **date**: Publication date and time
   - **description**: A brief description of your post
   - **tags**: Relevant tags (space-separated)
   - **categories**: Post categories
4. Write your content below the front matter using Markdown

## Markdown Examples

### Headings
Use `#` for headings (more `#` = smaller heading)

### Lists

**Unordered list:**
- Item 1
- Item 2
- Item 3

**Ordered list:**
1. First item
2. Second item
3. Third item

### Links and Images

[Link text](https://example.com)

![Image alt text](/path/to/image.jpg)

### Code

Inline code: `code here`

Code block:
```python
def hello_world():
    print("Hello, World!")
```

### Quotes

> This is a blockquote.
> It can span multiple lines.

### Emphasis

*Italic text* or _italic text_

**Bold text** or __bold text__

***Bold and italic*** or ___bold and italic___
